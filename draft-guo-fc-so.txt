               A Profile for Forwarding Commitments (FCs)
                         draft-guo-fc-so-latest

Abstract

   This document defines a standard profile for Forwarding Commitment
   (FC) objects for use with the Resource Public Key Infrastructure
   (RPKI).  A FC is a digitally signed object that provides a means of
   verifying that an IP address prefix is announced by AS a to AS b.
   When validate, an FC's eContent can be used for detection and
   mitigation of route hijacking.
